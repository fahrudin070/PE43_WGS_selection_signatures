# 05_fst

This folder contains curated scripts for weighted FST analysis, top 1% region extraction, annotation, gene matrix generation, and FST plotting.

Included steps:
- extraction of top FST windows
- annotation of FST windows to genes
- gene presence matrix generation
- Manhattan-style weighted FST plotting

Main tools:
- shell utilities
- R

Notes:
These scripts correspond to the final weighted FST workflow used in the manuscript.
