# 01_preprocessing

This folder contains curated scripts for reference indexing and read mapping.

Included steps:
- reference indexing
- per-sample read mapping
- master SLURM/Snakemake mapping runs for selected breeds

Main tools:
- bwa / bwa-mem
- samtools
- SLURM
- Snakemake

Notes:
These scripts represent the curated preprocessing components retained for manuscript documentation and reproducibility.
