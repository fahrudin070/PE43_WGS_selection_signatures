# 02_variant_dataset_construction

This folder contains curated scripts for variant calling, filtering, imputation, and final annotation.

Included steps:
- FreeBayes parallel calling
- bcftools-based SNP filtering
- Beagle imputation
- snpEff annotation

Main tools:
- FreeBayes
- bcftools
- Beagle
- snpEff

Notes:
These scripts represent the curated variant dataset construction workflow retained for manuscript documentation and reproducibility.
