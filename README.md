# PE43_WGS_selection_signatures

This repository contains curated scripts used in the manuscript on whole-genome sequencing and integrative selection signature analysis in Peranakan Etawah (PE) goats.

## Workflow overview
The repository is organized according to the manuscript workflow:

1. Preprocessing
2. Variant dataset construction
3. Population structure analysis
4. ROH analysis
5. Weighted FST analysis
6. Integrative ROH-FST selection analysis
7. Functional annotation
8. Visualization

## Notes
- Only curated final scripts used for manuscript-related analyses are retained here.
- Raw sequencing data, large intermediate files, BAM/VCF files, and temporary outputs are not included.
- Some analyses were run on HPC, whereas some downstream visualization steps (e.g. Circos plotting in R) were finalized locally using processed outputs from HPC analyses.
